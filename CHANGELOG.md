# Change Log

All notable changes to the "lira-free-theme" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.2.0] - 2025-01-18

### Added
- **Nord Theme**: New color theme variant based on the popular Nord color palette
  - `Lira Nord` - Arctic, north-bluish color scheme
  - `Lira Nord High Contrast` - Darker variant for improved contrast
  - `Lira Icons Nord` - Matching icon theme with Nord-styled folder icons
- 92 new folder icon SVG variants for Nord theme

### Changed
- Updated theme configuration to include Nord in all theme pickers

## [0.1.0] - Initial Release

### Added
- Initial release with 6 color theme variants (Teal, Graphene, Palenight, Ocean, Deepforest, Carbon)
- High contrast versions for all themes
- Matching icon themes for each variant
- Customizable accent colors (14 presets + custom hex)
- Settings panel with theme/icon/color customization
- Status bar menu for quick access to theme options
- Export/import configuration support
- Toggle options for borders, shadows, contrasted tabs, and more
